🏷️ 提示信息 Tooltip，指当鼠标悬停在图表上或者手指点按移动设备的某个数据点时，以**交互提示**信息的形式展示该点的数据，比如该点的值，数据单位等。

🎨  前往墨者学院 [AntV 设计 | 提示信息 Tooltip](https://www.yuque.com/mo-college/vis-design/vrxog6) 查看**设计指引**。
