#### Usage

There are two ways to configure labels

Method 1, pass in 'Boolean' to set whether to display a label.

```ts
label: false; // close label
```

Method 2, pass in _LegendCfg_ to configure the label as a whole.

```ts
label: {
  content: '标签'
}
```
