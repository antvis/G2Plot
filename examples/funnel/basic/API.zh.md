### 数据映射

#### data 📌

**必选**, _array object_

功能描述： 设置图表数据源

默认配置： 无

数据源为对象集合，例如：`[{ time: '1991'，value: 20 }, { time: '1992'，value: 20 }]`。

`markdown:docs/common/xy-field.zh.md`

#### compareField

**可选**, _string_

功能描述： 对比字段 。

默认配置： 无

#### transpose

**可选**, _boolean_

功能描述： 是否转置 。

默认配置： 无

`markdown:docs/common/meta.zh.md`

### 图形样式

#### dynamicHeight

**可选**, _boolean_

功能描述： 是否是动态高度 。

默认配置：`true`

#### annotation

**可选**, _boolean | string | Function_

功能描述： annotation 文本 。

默认配置：`true`
