_FacetTitle_ 配置如下：

| 参数名    | 类型                                     | 默认值 | 描述                |
| --------- | ---------------------------------------- | ------ | ------------------- |
| offsetX   | number                                   | -      | x 方向偏移          |
| offsetY   | number                                   | -      | y 方向偏移          |
| style     | [ShapeAttrs](/zh/docs/api/graphic-style) | -      | 文本样式            |
| formatter | (val: any) => any                        | -      | 回调配置 title 内容 |
